##### Here's features included in this project

- Get posts and preview data table

- Add new post to table

- Update post

- Delete post

- Validation Inputs error

- Handle Error from API

1. React.js
2. Typescript
3. Ant Design
4. React router
5. Formik
6. Axios
7. React-toastify
8. Tailwind css
